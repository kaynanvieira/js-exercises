# JavaScript Logic Exercises

Lista de exercícios desenvolvidos para praticar lógica de programação utilizando JavaScript.

## Exercícios

1. Soma de dois números
2. Maior número
3. Verificação de número primo
4. Números pares em um intervalo
5. Sequência de Fibonacci
6. Conversor Celsius → Fahrenheit

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript
